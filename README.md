# PassPORT-PassPoseOptimalRouteTracker
PassPORT is an sports climbing coach that uses Computer Vision to analyze your climbing posture and movements.

컴퓨터 비전(CV) 기술을 활용해 클라이머의 자세를 분석해 주는 코칭 프로젝트입니다. 

클라이밍에서는 힘만큼이나 효율적인 자세와 완등 공략법이 중요합니다.
하지만 혼자 영상을 찍어볼 때는 내 자세의 어디가 잘못되었는지 눈으로만 확인하기 어렵습니다.


PASSPORT는 자세 추정을 통해 영상 속 클라이머의 관절 위치와 무게 중심의 이동을 실시간으로 추적합니다. 벽 위에서 어떤 동작이 비효율적이었는지, 왜 떨어졌는지를 정밀하게 분석하여 다음 난이도 레벨(V-grade)로 통과할 수 있는 자세을 제시합니다.

---

### 스포츠 클라이밍(볼더링) 이란

볼더링은 로프나 하네스 등의 장비 없이, 약 5미터 이하의 낮은 인공 벽이나 자연 바위를 오르는 클라이밍의 한 종류입니다. 안전장비 대신 바닥에 두꺼운 매트를 깔고 진행하며, 클라이머가 풀어야 할 하나의 루트를 **문제를 푼다**라고 표현합니다.

지구력이 중요한 로프 클라이밍과 달리, 볼더링은 **순간적인 힘과 정밀한 자세, 그리고 문제 해결 능력**이 핵심입니다. 체중을 어디로 이동시키는지, 골반을 벽에 얼마나 밀착하는지, 발을 어떤 각도로 딛는지와 같은 아주 미세한 동작의 차이가 완등과 추락을 결정짓습니다.

### 볼더링 난이도 체계: V-스케일(V-Scale)

볼더링은 전 세계적으로 **V-스케일(V0 ~ V17)**이라는 난이도 체계를 주로 사용합니다. 
숫자가 높아질수록 잡기 힘든 홀드가 나오고, 벽의 경사가 가팔라지며, 요구되는 동작의 난이도가 기하급수적으로 어려워집니다.

| 난이도 | 등급 (Grade) | 특징 및 요구 능력 |
| :--- | :--- | :--- |
| **V0 – V2** | **초급 (Beginner)** | 기본적인 근력과 기본적인 발 디디는 법을 배우는 단계 |
| **V3 – V5** | **중급 (Intermediate)** | 효율적인 자세, 코어의 힘, 그리고 자신만의 '베타(공략)'가 필요한 단계 |
| **V6 – V9** | **상급 (Advanced)** | 다이노(점프) 같은 다이내믹한 무브와 강력한 손가락 힘, 미세한 조절이 필요한 단계 |
| **V10+** | **엘리트 (Elite)** | 신체적 한계를 뛰어넘는 힘과 완벽에 가까운 기술이 결합된 단계 |
